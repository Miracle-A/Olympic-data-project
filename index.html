Azure Data Engineering Project

This project showcases how I used Azure Data Factory, Databricks, and Synapse Analytics to process Olympic data.

Architecture Diagram

Project Overview

  • Extracted data using Azure Data Factory.
  • Transformed data with Databricks (PySpark).
  • Loaded data into Synapse Analytics for querying.
  • Generated insights and visualizations from the transformed data.
