Project Overview

This project showcases an end-to-end data pipeline built using Azure services, including Azure Data Factory, Databricks, and Synapse Analytics. It analyzes the Tokyo 2020 Olympics data to extract meaningful insights and demonstrate cloud-based data engineering skills.

View on GitHub

SQL Queries and Insights

  • Top Countries by Gold Medals: SQL script to analyze countries with the most gold medals.
  • Average Male/Female Participation: Query to show the average gender participation across sports.
  • Total Medals by Country: Query to display the total medals won by each country.

Challenges and Learnings

This project provided hands-on experience in:

  • Building data pipelines using Azure Data Factory, Databricks, and Synapse Analytics.
  • Writing and optimizing SQL queries for data analysis.
  • Managing cloud resources and storage efficiently.
